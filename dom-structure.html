Architecture

Website DOM Structure

This page outlines the Document Object Model (DOM) structure used across the fashion house website and shared components.

High-Level DOM Tree

html
  head
    meta*
    title
    script (tailwind, config, head inject)
  body
    div (page wrapper)
      section#header (injected from inc/header.html)
      main
        section* (page-specific content)
      section#footer (injected from inc/footer.html)
      script (js/custom.js)

Shared Components

  • Header Include: Shared navigation links and theme toggle.
  • Footer Include: Shared marquee and contact/footer info.
  • JS Layer: Header/footer injection, theme toggle, greeting, product filtering, form validation.
  • Styling: External styles from css/fonts.css and css/style.css.
