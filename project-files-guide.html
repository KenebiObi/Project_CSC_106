Project Documentation

Complete Project File Guide

This guide reflects the current project structure after the major page, content, and asset updates made across the site.

Project Structure Summary

  • Root HTML pages: 7 files
  • Shared includes: 2 files in inc/
  • JavaScript files: 3 files in js/
  • CSS files: 2 files in css/
  • Image assets: 27 files in img/
  • Tracked project files: 44 files

Root HTML Files

index.html

Home page. Contains the hero banner, greeting text, heritage story section, collection preview, and main entry points into the collections and contact pages.

collections.html

Product showcase page. Displays a 9-item product grid split across Yoruba, Hausa, and Igbo categories, with filter buttons powered by filterItems() in js/custom.js.

board.html

Board of Trustees page. Shows the current leadership team in multiple responsive card grids with hover overlays, role labels, and individual profile images.

events.html

Events page. Includes a hero section with live countdown placeholders, a VIP invitation call to action, and the 2026 showcase schedule for Lagos, Abuja, and Enugu.

contact.html

Contact and appointments page. Contains the private consultation form, validation error states, success state, brand/location information, and booking-related messaging.

dom-structure.html

Documentation page that explains the shared DOM layout used across the site, including injected header/footer sections, scripts, and reusable structure.

project-files-guide.html

This documentation page. Summarizes the current purpose of each page, include, script, stylesheet, and image group in the project.

Shared Includes (inc/)

inc/header.html

Shared navigation header. Includes links to the main pages, documentation pages, a theme toggle button, and a mobile menu icon placeholder.

inc/footer.html

Shared footer. Contains the animated event marquee, brand summary, explore links, contact details, and bottom legal links. It is loaded dynamically into every page.

JavaScript Files (js/)

js/head.js

Injects the shared stylesheet links for css/fonts.css and css/style.css into the document head after page load.

js/tailwind-config.js

Defines the shared Tailwind theme, including the project primary color, background colors, display font, radius scale, and class-based dark mode setup.

js/custom.js

Main behavior file. Loads the header and footer includes, handles collection filtering, updates the home-page greeting, validates the booking form, manages the dark/light theme toggle, and runs the event countdown timer.

CSS Files (css/)

css/fonts.css

Imports the Material Symbols icon set and the Manrope font family, then applies the base type and icon rendering settings used throughout the project.

css/style.css

Holds the custom global styles that sit beside Tailwind utilities, including fallback body colors, the brand text color helper, marquee animation, and active collection filter state.
